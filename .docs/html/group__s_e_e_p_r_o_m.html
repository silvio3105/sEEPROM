Detailed Description

Simple EEPROM driver. It provides functions for easy access to EEPROM on STM32 & nRF52 chips. Check chip datasheet for EEPROM start address(es) and length(s). It is possible to create multiple objects which represents different parts of EEPROM(eg., EEPROM for device config, EEPROM for device history etc.).
